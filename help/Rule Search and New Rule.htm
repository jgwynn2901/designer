Rule Search and New Rule

If you need to find a particular rule, you may want to use Rule Search (Rule Search examines the rule table for a rule.). You do a Rule Search to find the rule and copy the rule to use in your call flow.

You can also use New Rule (A new rule is an original rule added to the rule table.) if you can not find a rule that meets your needs.

Click Navigation Details for information on how to get to Rule Search and New Rule.

Navigation Details

1. From the navigation bar, choose "Maintenance."

2. Click the "+" sign next to the "Attribute Related" folder.

3. Select the "Rules" page from the Maintenance tree.

Common Tasks

These Common Tasks are used when using Rule Search and New Rule. Click the green text for detailed information on each task.

Rule Search

Rule Search

Enter the Rule Search information.

  • Select Type of Rule and enter Rule Text.

  • Enter Rule ID.

  • Select Type of Search.

The default is "Begins With." "Begins With" means the Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the "Search" button.

  • Select results from results list and click the "Details" tab.

  • Click the "Make Copy" button.

Note

You can also delete the rule by clicking the "Delete" button.

New Rule

  • Click the "New" button on the Rule Search page.
