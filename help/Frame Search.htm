Frame Search

If you need to find a particular frame you may want to use Frame Search (Frame Search examines the frame table for a frame.). You can also delete a frame.

Click Navigation Details for information on how to get to Frame Search.

Navigation Details

1. From the navigation bar, choose "Maintenance."

2. Click the "+" sign next to the "Call Flow Related" folder.

3. Select "Frame" page on the Maintenance tree.

Common Task

This Common Task is used when doing a frame search. Click on the green text for detailed information on this task.

Frame Search

Frame Search

Enter the frame search information.

  • Enter A.H.S. ID and Client Node ID.

  • Enter Frame ID and Name.

  • Select Line of Business and enter Title.

  • Select Type of Search

The default is "Begins With." "Begins With" means the Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Select result from result list and click the "View" button.

Note

You can also delete the frame by clicking the "Delete" button.
