Lookup Type Search and New Lookup Type

If you need to find a particular Lookup Type, you may want to use Lookup Type Search (Lookup Type Search examines the Lookup Types table for a Lookup Type.) You do a Lookup Type Search to find a Loopkup Tye and copy the lookup type to use in your call flow.

You can also use New Lookup Type (A new Lookup Type is an original Lookup Type added to the lookup Type tabe.) if you can't find an attribute that suits your needs.

Click Navigation Details for information on how to get Lookup Type and New Lookup Type.

Navigation Details

1. From the navigation bar, choose "Maintenance."

2. Click the "+" sign next to the "Attribute Related" folder.

3. Select the "Lookup Types" page from the Maintenance tree.

Common Tasks

These Common Tasks are required when using Lookup Type Search and New Lookup Type. Click the green text for detailed information on each task.

Lookup Type Search

Lookup Type Search

Enter Lookup Type Search information.

  • Enter Name and Lookup Type ID.

  • Select Type of Search.

The default is "Begins With." "Begins With" means the Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the "Search" button.

  • Select results from results list and click the "Details" tab.

  • Click the "Make Copy" button.

Note

You can use the "Back" button to return to Lookup Type Search.

New Lookup Type

New Lookup Type

  • Click the "New" button on the Lookup Type Search page.

  • Enter Name and click "Save."
