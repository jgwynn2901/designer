Filtering Routing Plans

You have several choices at the routing plan level on the Business Entity Summary page. You can filter routing plans.

If you need a customized list of routing plans, you may want to filter routing plans. A filter blocks certain information and allows other information to pass based on criteria you enter.

Click Navigation Details for information on how to get to Filtering Routing Plans.

Navigation Details

1. From the Navigation bar, choose Hierarchy.

2. Click the "+" sign next to FNS in the Hierarchy tree.

3. Select the client in the Hierarchy Tree to go to the Business Entity Summary page in the right frame.

4. Click the "+" sign next to Routing Plans on the Business Entity page.

Common Task

This Common Task is used when Filtering Routing Plans. Click the green text for detailed information on the task.

Filtering Routing Plans

Filtering Routing Plans

Click the "Filter" button on the Routing Plans bar.

Enter or select Routing Plans information.

  • Select Destination Type from drop down list.

  • Select Input System Name from drop down list.

  • Select Line of Business and Enabled from drop down lists.

  • Select Type of Search

The default is "Begins With". "Beings With" means Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.
