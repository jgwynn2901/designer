Routing Plan Basics

Routing Plan Basics contains the fundamentals of routing plans. These topics provide the essentials of building routing plans and using routing plan components in step by step procedures.

If you need to know where to begin to build routing plans read "Building Routing Plans." You may want to add a transmission sequence step for a routing plan as described in "Adding Transmission Sequence Steps for Routing Plan".

You may want to add routing plan components such as output items, output definitions, output pages and overflow pages. "Adding Output Items" describes the alternatives for adding output items. "Adding Output Definitions" describes the options for adding output definitions. You have several options for output pages and overflow pages as detailed in "Working with Output Pages" and "Working With Overflow Pages." "Reviewing Output Field Listing and General Information" provides output field information and general information.

You have several choices at the flow level. "Editing and Refreshing Routing Plans" explains how to edit and refresh routing plans. "Copying and Pasting Routing Plans" details how to copy and paste routing plans. "Filtering Routing Plans" details how to filter routing plans.
