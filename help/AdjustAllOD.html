Adjust All

1. Adjust X-Coordinate:
  a.)Negative number moves fields LEFT.
  b.)Positive Number moves fileds RIGHT.

2. Adjust Y-Coordinate:
  a.)Negative number moves fields UP.
  b.)Positive Number moves fileds DOWN.

3. Adjust Width:
  a.)Negative number decreases field width.
  b.)Positive number increase field width.

4. Adjust Height:
  a.)Negative number decreases field height.
  b.)Positive number increases field height.

Notes: 6 units = approx. 1 pixel

Warning:
Adjust all cannot be undone,
user must cancel changes.
