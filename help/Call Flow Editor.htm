Call Flow Editor

A call flow consists of frames, attributes and rules. You use the Call Flow Editor to create those frames, attributes and rules.

 

Call Flow Editor Access

Call Flow Editor Access From Maintenance:

  • From the Navigation bar, choose Maintenance.

  • Click the "+" sign next to Calll Flow Releated in the left frame.

  • Click Call Flow in the left frame.

  • Use Call Flow Search to find and select the appropriate call flow.

  • Right click the mouse in the Call Flow Frames area.

  • Select Visual Editor from popup menu.

Call Flow Editor Access From Hierarchy:

  • From the Navigation bar, choose Hierarchy.

  • Click the "+" sign next to FNS in the Hierarchy Tree.

  • Select the client in the Hierarchy Tree to display the Business Entity Summary page in the right frame.

  • Click the "+" sign next to Account Call Flows on the Business Entity Summary page.

  • Select call flow from list of call flows.

  • Click the Edit button on the Account Call Flows bar.

  • Right click the mouse in the Call Flow Frames area.

  • Select Visual Editor from the popup menu.

 

Call Flow Editor Frame Options

A frame is the screen the call representative uses to get the requested information. For example, "Caller Information" is a frame for collecting caller information and "Employee Information" is a frame for collecting employee information.

The Call Flow Editor gives you four frame options:

  • New Frame

You create a new or blank frame from scratch. Use this option if your frame requires a lot of original information that does not exit on other frames.

  • Attach Existing Frame To This Call Flow

You attach or connect an existing frame from the frame database to this call flow. If you use this option and make changes you are changing the frame in the frame database. The frame in the frame database is most likely used by other call flows. Therefore use this option if you can use the frame as is.

  • Copy Frame As New

You make a copy of the frame in the frame database. If you use this option and make changes the changes effect only your copy of the frame not the original frame in the frame database.

  • Detach This Frame From Current Call Flow

You detach the frame from the current call flow. This option does not delete the frame, but does detach the frame from the current call flow.

In addition, the Call Flow Editor has six tabs. The "General" tab contains general call flow information such as name and title. You use the "Layout" tab to add attributes to the frame and see the frame visually. Attributes are "fields" such as "First Name" or "Last Name." You use the "Rules" tab to add rules to the frame.

The "SQL Data" tab enables you to manipulates SQL Data. The "Listing" tab contains Attribute Instance Listing for the frame. In addition the "Frame Order" tab contains Frame Order information.
