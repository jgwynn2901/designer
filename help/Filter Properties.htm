Filter Properties

Filter Properties applies a filter to the Hierarchy Tree. This creates a custom view on the Hierarchy Tree enabling you to find information quickly. The filter blocks unwanted information and displays on the information you requested in the Hierarchy Tree.

Click on Navigation Details for information on how to get to Filter Properties.

Navigation Details

1. On the Designer navigation bar, click Hierarchy.

2. Click the "+" sign next to FNS on the Hierarchy Tree.

3. Select a client, risk location or account on the Hierarchy Tree.

4. Right click the mouse on the client, risk location or account.

  • Click "Filter Properties" on the popup menu.

  • Enter or select Filter Properties information.

  • Select Type of Search.

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Use Filter Field is checked off by default.

  • Click the Search button.

  • Select result from results list.

  • Click the Apply button.

The Hierarchy Tree refreshes and displays the appropriate node. A funnel icon appears next to the node indicating the node is filter enabled.

Notes

Temporary Filter or Permanent Filter

Whether the filter is temporary or permanent depends on a setting in the Settings page dialog box. If you want a temporary filter:

  • On the Designer Navigation bar, choose Settings.

  • Make sure the "Filters" check box is unmarked.

  • Click "Save Settings" button on the Settings Page dialog box.

  If you want a permanent filter:

  • On the Designer Navigation bar, choose Settings.

  • Check "Filters" on the Settings page dialog box.

  • Click "Save Settings" button on the Settings Page dialog box.

 

Must Exclude And Must Include

You can broaden or narrow the filter excluding or including parameters. If you want to add a parameter use the "Must Include" text box. If you want to exclude a parameter use the "Must Exclude" text box.

Clear Filter

Clears the filter and returns the Hierarchy Tree to normal.

Clear

Clears the filter parameters.

Close

Closes the filter dialog box.
