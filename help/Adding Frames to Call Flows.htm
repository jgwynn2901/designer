Adding Frames to Call Flows

Once you have a call flow you'll want to add frames (A frame is a screen or page requesting information such as "Caller Information" or "Loss Location".) to the call flow. You can either create a new frame, copy frame as new or attach existing frame to this call flow. Whether you want to create a new frame, copy frame as new or attach existing frame to this call flow depends on how you want to use the frame.

If your frame requires a large amount of original information not existing on other frames, you may want to create a new frame (A new frame is an original frame added to the frame database.). If your call flow requires a great deal of the same information found on other frames, you may want to copy frame as new (A copy frame as new is a duplicate of a frame in the frame database. Depending on how you got the call flow changes may or may not effect the frame in the frame database. If you attached or copied the call flow changes still effect the frame in the frame database. If you created a new call flow changes effect only your copy of the frame and not the original frame in the frame database.). If you plan to use the frame as is, you may want to attach existing frame to call flow (An attached frame is an existing frame in the database used in other call flows.).

Attaching an existing frame to a call flow has important consequences because you are connecting to an existing frame in the frame database. For example if you are attaching to a frame such as "Caller Information" you are connecting to the "Caller Information" frame in the database.

If you make changes to this attached frame you are changing the frame in the frame database. This effects other call flows as well because the attached frame is a shared frame used in multiple call flows. Therefore you should use a attach existing frame to this call flow only if you can use the shared frame as is. A shared frame such as "Caller Information" should already have all the information required so you can use it without changing it.

If you attached to a frame you really didn't want to attach to, you may want to detach this frame from current call flow (An detached frame is separated from the frame in the frame database.). This does not delete the frame from the frame database. The frame still exists. This option detaches the frame from the current call flow.

Navigation Details

1. From the Navigation bar, choose Hierarchy and click the "+" sign next to FNS in the Hierarchy Tree.

2. Select the client in the Hierarchy Tree to display the Business Entity Summary page in the right frame.

3. Click the "+" sign next to Account Call Flows on the Business Entity Summary page.

4. Select call flow from list of call flows and click the Edit button on the Account Call Flows bar.

5. Right click the mouse on the call flow in the Call Flow Frames section and select Visual Editor from the popup menu.

Common Tasks

These Common Tasks are used when Adding Frames to Call Flows. Click the green text for detailed information on each task.

New Frame

New Frame

Click the New Frame icon on the Call Flow Name bar on the Call Flow Editor.

Enter or select the New Call Flow Frame information below.

  • Enter Name and Title.

  • Enter Help String.

The Help String is the information in the Tips frame in the call flow screen. The Tips frame contains suggestions for taking the information from the caller. For example, the Tips frame in the Caller Information screen could read "(If the caller is someone who has called previously, perform a caller lookup to access our database of clients.)"

  • Enter Description, Attribute Prefix, Type and Maximum Results Rows.

Click the Save button.

Note

The new frame you created appears in the Call Flow Frames drop down list on the Call Flow Name bar on the Call Flow Editor. You can also minimize the Call Flow Editor to see the new frame in the Call Flow Frames area.

Copy Frame As New

Copy Frame As New

You must do a Frame Search to find the frame before you can Copy Frame As New.

Click the Attach Existing Frame To This Call Flow icon on the Call Flow Name bar on the Call Flow Editor.

Enter or select Frame Search information below.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the Search button and select the appropriate result from the results list.

Example

  • Click the Select button and the Save button.

Click the Copy Frame As New icon on Frame Name bar on the Call Flow Editor.

Click OK to the dialog box asking "Are you sure you want to copy this frame? Copying this frame will create a new unique instance of this frame and the current frame will be detached."

Click the Save button.

Example

Note

The frame you copied as new appears in the Call Flow Frames drop down list on the Call Flow Name bar on the Call Flow Editor. You can also minimize the Call Flow Editor to see the frame you copied as new in the Call Flow Frames area.

Attach Existing Frame To This Call Flow

Attach Existing Frame To This Call Flow

Click the Attach Existing Frame To This Call Flow icon on the Call Flow Name bar on the Call Flow Editor.

Enter or select Frame Search information below.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the Search button.

Example

  • Select the appropriate result from the results list and click the Select button.

Click the Save button.

Note

The frame you attached to your call flow appears in the Call Flow Frames drop down list on the Call Flow Name bar on the Call Flow Editor. You can also minimize the Call Flow Editor to see the frame you attached to your call flow in the Call Flow Frames area.

Detach This Frame From Current Call Flow

Detach This Frame From Current Call Flow

Click the Detach This Frame From Current Call Flow icon on the Frame Name bar on the Call Flow Editor.

Click OK on the "Are you sure you want to detach frame?" dialog box.

Note

The frame you detached disappears in the Call Flow Frames drop down list on the Frame Name bar on the Call Flow Editor. You can also minimize the Call Flow Editor and notice the detached frame is gone from the Call Flow Frames area.
