Designer and Call Flows

The Designer is a tool for designing business entities, data and processing rules for FNSNet. It enables business analysts to create and edit Call Flows, Branching Assignments, Routing Plans, Output Definitions and Rules. These items created by business analysts are used to determine where and what data is send in FNSNet.

The Designer is a GUI front end for creating call flows. The Call Flow Editor provides a central location for doing the tasks necessary to create a call flow.
