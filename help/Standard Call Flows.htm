Standard Call Flows

You should use a Standard Call Flow ( A standard call flow is a pre-constructed call flow.) when building call flows. Standard Call Flows are similar to "templates" in Microsoft Word containing pre-made content that can be modified to meet your needs. For example, the "Standard Workers Compensation Call Start" call flow named "zSC WOR CALL START" and the "Standard Workers Compensation Call Flow named "zSC WOR CALL FLOW" requests the information required to correctly file a Workers Compensation Claim.

Standard Call Flows are time savers because a standard call flow has most of the information you'll need to build a call flow. You copy the standard call flow to the clipboard and paste the standard call flow from the clipboard to its new location. Because it is a copied call flow (A copied call flow is a duplicate of a call flow in the call flow table.) you can change the standard call flow by editing it.
