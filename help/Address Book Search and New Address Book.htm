Address Book Search and New Address Book

If you need to use the same item over and over again, you may want to use the Address Book (The Address Book is used to insert frequently used items into a call flow.). The Address Book saves you time by storing selected items that are used repeatedly. You do an Address Book search to find the item and copy the item to use it.

You can also add a new item to the Address Book so that you and others can use it in the future.

Click Navigation Details for information on how to get to Address Book Search and New Address Book.

Navigation Details

1. From the navigation bar, choose "Maintenance."

2. Click the "+" sign next to the "Call Flow Related" folder.

3. Select "Address Book" page on the Maintenance tree.

Common Tasks

These Common Tasks are used when using Address Book Search and Address Book. Click the green text for detailed information on each task.

Address Book Search

Address Book Search

Enter the Address Book Search information.

  • Enter Description.

  • Enter Address Book ID, Call Flow ID and Name.

  • Select Type of Search.

The default is "Begins With." "Begins With" means the Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the "Search" button.

  • Select results from results list and click the "Details" tab.

  • Click the "Make Copy" button.

Note

You can also delete the Address Book item by clicking the "Delete" button.

New Address Book

New Address Book

  • Click the "New" button on the Address Book Search page.

  • Use the Call Flow Search button to find the Call Flow ID.

  • Enter Name and Description.

  • Click the "Save" button.

Note

The "Back" button returns you to the Address Book Search page.
