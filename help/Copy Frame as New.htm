Copy Frame As New

You must do a Frame Search to find the frame before you can Copy Frame As New.

1. Click the Attach Existing Frame To This Call Flow icon in the upper left corner of the Call Flow Editor.

2. Enter or select Frame Search information below.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the Search button.

  • Select the appropriate result from the results list.

  • Click the Select button.

3. Click the Save button.

4. Click the Copy Frame As New icon in the upper left corner next to Frame Name in the Call Flow Editor.

5. Click OK to the dialog box asking "Are you sure you want to copy this frame? Copying this frame will create a new unique instance of this frame and the current frame will be detached.

6. Click the Save button.

Note

Copy Frame As New detach the existing frame meaning your call flow contains only the frame copied as new. The Copy Frame As New Frame appears in the Call Flow Frames drop down list at the top of the Call Flow Editor. You can also minimize the Call Flow Editor to see the Copy Frame As New Frame in the Call Flow Frames area.
