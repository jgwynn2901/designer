Adding Attributes to Frames

Once you have a call flow and frames, you'll want to add attributes to the frames using the Layout tab on the Call Flow Editor. You have several choices when adding attributes (An attribute is a field such as "First Name" and "Zip Code.") to frames.

Find Attributes and Create Attributes

Whether you want to Find Attributes or Create Attributes depends on how you want to use the attribute. If your attribute requires a great deal of the same information found on other attributes, you should use Find Attributes (Find Attribute is an attribute search based on criteria you enter.) to use an existing attribute. If your attribute requires a large amount of original information not found on other attributes, you should use Create Attributes (Create Attribute creates a new or original attribute added to the attribute database) to make an attribute that fits your needs.

Instance Properties and Attribute Properties

Whether you want to use Instance Properties or Attribute Properties depends on how you want to use the properties of the attribute. If you plan on making changes to the properties, you may want to use Instance Properties (An Instance Properties is essentially a copy or duplicate of attribute properties. If you change Instance Properties these changes effect only your "duplicate" of the properties.).

Instance Properties allows you greater flexibility because you are using a duplicate or copy of the attribute's properties. This allows you to modify the copy of the attribute properties to suit your needs. Therefore you should use Instance Properties if you are going to modify properties.

If you plan on using the Attribute Properties as is, you may want to use Attribute Properties (An Attribute Properties are the properties or characteristics of an attribute. If you change Attribute Properties changes directly effect the attribute which is often shared.).

Attribute Properties has important consequences because you are using the actual properties of an attribute. If you make changes to the Attribute Properties you are changing the properties of the attribute in the database. This effects all users of the attribute. Therefore you should use Attribute Properties only if you do not want to change properties.

Override and Remove

You have the option to Override or Remove. If you want need to override or change an attribute property, you may want to use Override (An Override overrules an attribute property.). If you need delete an attribute, you may want to use Remove (Remove eliminates the attribute from the attribute database.).

Layout Tab Icons and Popup Menu

The Layout Tab contains icons you use when adding attributes. These icons are Attribute Search, Attribute Properties, Save, Undo, Zoom, Override and Remove. The Popup Menu contains choices that effect the way you view information on the Layout Tab. These choices are Show Select, Show All, Show Attributes and Show Samples.

Navigation Details

1. From the Navigation bar, choose Hierarchy and click the "+" sign next to FNS in the Hierarchy Tree.

2. Select the client in the Hierarchy Tree to display the Business Entity Summary page in the right frame.

3. Click the "+" sign next to Account Call Flows on the Business Entity Summary page.

4. Select call flow from list of call flows and click the Edit button on the Account Call Flows bar.

5. Right click the mouse on the call flow in the Call Flow Frames section and select Visual Editor from the popup menu.

6. Click the Layout tab on the Call Flow Editor.

Find Attributes and Create Attributes

Find Attributes and Create Attributes

Find Attributes

Click the Attribute Search icon on the Layout tab bar.

  • Enter Name and Caption.

  • Select Input Type from drop down list.

  • Enter Description, Help String and Attribute ID.

The Help String is information in the Tips Frame in the call flow screen. The Tips Frame contains suggestions for taking the information from the caller when the call representative uses the attribute.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the Search button.

Example

  • Select appropriate result from results list.

  • Click the Select button.

The attribute appears in the Attribute Name List drop down box.

It also appears in the Call Fow Editor.

Example

Create Attributes

Click the Attribute Search icon on the Layout tab bar.

  • Click the New button on the Attribute Search dialog box.

  • Enter Name and Caption.

  • Enter Unknown Value and select Input Type from drop down list.

  • Enter Text Length, Entry Mask and Default Value.

  • Enter Description and Help String.

The Help String is information in the Tips Frame in the call flow screen. The Tips Frame contains suggestions for taking the information from the caller when the call representative uses the attribute.

  • If applicable attach/detach rules to attributes.

Note

Creating a new attribute requires a knowledge of FNS Net architecture and data flow. Please consult with a developer before creating a new attribute.

Instance Properties and Attribute Properties

Instance Properties and Attribute Properties

Instance Properties

Right click the mouse inside the attribute and select "Instance Properties" from the popup menu.

Example

  • Enter Caption and Sample Value.

  • Enter Xpos, Y Pos, Width, Height, Sequence and select Auto Type.

  • Enter LU Column Name and LU Storage Name.

  • If applicable, check LU Display flag, LU Storage Flag, Mandatory, Valid Value and Spell Check.

  • Enter Entry Mask and Default Value.

  • Enter Unknown Value and Text Length.

  • Enter Help String and Description.

The Help String is information in the Tips Frame in the call flow screen. The Tips Frame contains suggestions for taking the information from the caller when the call representative uses the attribute.

  • Attach/Detach rules if applicable.

  • Click the Ok button.

Attribute Properties

Right click the mouse inside the attribute and select "Attribute Properties" from the popup menu.

Example

  • Check the Edit box.

  • If applicable check Valid Value? and Spellcheck?

  • Enter Unknown Value and select Input Type from drop down list.

  • Enter Text Length, Entry Mask and Default Value.

  • Enter Description and Help String.

The Help String is information in the Tips Frame in the call flow screen. The Tips Frame contains suggestions for taking the information from the caller when the call representative uses the attribute.

  • Attach/Detach rules if applicable.

  • Click the Save button.

Override and Remove

Override and Remove

Override

Click the Override icon on the Layout tab bar or right click inside and select "Override" from the popup menu.

New Attribute Over Ride

Select "New" on the Attribute Over Ride Listing bar.

  • Select Property Name from drop down list.

  • Enter Sequence.

  • Click the Attach Rule icon to find a rule to attach to the override or create a new rule to attach to the over ride.

To Find Rule

  • Select Type.

  • Enter Rule Text and Rule ID.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the Search button.

  • Select result from results list.

  • Click the Select button.

To Create New Rule

  • Click the New button on the Rule Search box.

  • Select Rule Type from drop down list.

  • Enter Rule Text.

  • Click the Save button.

  • Enter Attribute Over Ride data.

    • Click Save button on the Attribute Over Ride Details dialog box.

    Edit Attribute Over Ride

    Select the appropriate attribute over ride and click the "Edit" button on the Attribute Over Ride Listing box.

    Edit or change the Attribute Over Ride Information below.

    • Property Name

    • Sequence

    • Over Ride Rule

    • Attribute Over Ride Data

    Click the Save button.

    Delete Attribute Override

    Select the appropriate attribute over ride and click the "Delete" button on the Attribute Over Ride Listing box.

    Click OK on the "Are you sure you want to delete this over ride?" dialog box.

    Remove

    Click the Remove icon on the Layout tab bar or right click inside the attribute and click "Remove" from the popup menu.

    Layout Tab Icons and Popup Menu

    Layout Tab Icons and Popup Menu

    Layout Tab Icons

    • Attribute Search icon accesses attribute search.

    • Properties icon accesses Instance Properties or Attribute Properties.

    • Save icon saves any changes to the Designer.

    • Undo icon undoes the last action.

    • Zoom icon zooms in for a close or enlarged view of the frame.

    • Override icon overrides an attribute property.

    • Remove icon removes the attribute from the frame.

    Popup Menu

    Show Select

    Displays the attribute selected. Right click the mouse inside the frame and select "Show Select" from the popup menu.

    Show All

    Displays all the attributes on the frame. Right click the mouse inside the frame and select "Show All" from the popup menu.

    Show Attributes

    Displays attributes such as "CLAIM:CLAIM_NUMBER". Right click the mouse inside the frame and select "Show Attributes" from the popup menu.

    Show Samples

    Displays the attributes as they would appear to the user. Right click the mouse inside the frame and select "Show Samples" from the popup menu.
