Attribute Search and New Attribute

If you need to find a particular attribute, you may want to use Attribute Search (Attribute Search examines the attribute table for an attribute.). You do an Attribute Search to find the attribute and copy the attribute to use in your call flow.

You can also use New Attribute (A new attribute is an original attribute added to the attribute table.) if you can't find an attribute that suits your needs.

Click Navigation Details for information on how to get Attribute Search and New Search.

Navigation Details

1. From the navigation bar, choose "Maintenance."

2. Click the "+" sign next to the "Attribute Related" folder.

3. Select the "Attributes" page from the Maintenance tree.

Common Tasks

These Common Tasks are required when using Attribute Search and New Attribute.

Attribute Search

Attribute Search

Enter the Attribute Search information.

  • Enter Name and Description.

  • Select Input Type.

  • Enter Description, Help String and Attribute ID.

  • Select Type of Search.

The default is "Begins With." "Begins With" means the Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the "Search" button.

  • Select results from results list and click the "Details" tab.

  • Click the "Make Copy" button.

Note

You can also delete the attribute by clicking the "Delete" button.

New Attribute

New Attribute

  • Click the "New" button on the Attribute Search page.
