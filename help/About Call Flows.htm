About Call Flows

Call Flows are used to collect claim information by the call representative from the caller. It is part of FNSNet.

The call representative uses call flows to obtain claim information from the caller. The call flow (A call flow is a group of screens that collect or lookup data as the call representative is taking the call.) requests claim information required to complete a particular type of call. It is specific to a line of business. For example the "Workers Compensation" call flow requests information required to correctly file a "Worker's Compensation Claim." The "Commercial Liability" call flow requests information required to correctly file a "Commercial Liability" claim.

This is a screen from a call flow that a call representative might see when using a call flow to enter a call.
