About Designer Help

In this help, you will find explanations of the many features in the Designer. It is designed to display in Microsoft's Internet Explorer 4.0 browser. The Designer Help is created for business analysts.

Tips for Using Designer Help

You can use a variety of navigational features in the Designer Help including:

  • Topics Links are jumps to topics shown in a frame. When you click a Topic Link the destination topic is displayed in the frame.

Topic Link Sample: Standard Call Flows

  • Popup Windows Links are jumps to topics shown in popup windows. When you click a Popup Window Link the destination topic opens in a popup window inside the frame you have opened. When you finish reading the information you click inside the popup window to close it.

Popup Window Link Sample: About Call Flows

  • View Previous Topic enables you to see the prior topic you opened. Right click inside the topic frame and select "Back" from the popup menu.

  • Text Only Popups display definitions in small popup windows. When you finish reading the information you click inside the text only popup window to close it.

Text Only Popup Sample: The call flow requests claim information required to complete a particular type of call.

  • Drop down hotspots are display additional information in a drop down format. When you click on the hotspot additional information is shown. When you finish reading the information click on the hotspot again to close it.

Drop Down Hotspot Sample: Call Flow Example

  • Related Topics button brings you to a related topic. When you click a Related Topics button a popup menu opens and you select the topic you want to see.

Related Topics Sample:

You have two choices for printing in Designer Help.

  • Click "File" on the Microsoft Explorer menu bar and a drop down menu opens. Select "Print" on the drop down menu.

Click "OK" on the print dialog box.

  • Right-click the mouse inside the topic frame and a popup menu opens. Select "Print" on the popup menu.

Click "OK" on the print dialog box.
