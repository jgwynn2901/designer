Account Call Flows

A call flow is a group of screens that collect or lookup data as you are taking the call. The call flow requests the claim information required to compete a particular type of call. It is specific to a line of business and is unique for each client. An account call flow is a call flow for a particular account.

For example the "Workers Compensation" call flow requests information required to correctly file a "Worker's Compensation Claim." The "Commercial Liability" call flow requests information required to correctly file a "Commercial Liability" claim.

You can create a new call flow or edit a call flow. You can also copy a call flow to the clipboard and paste the call flow from the clipboard into another Business Entity Summary page. Filter Call Flow enables you to create a custom list of call flows. In addition, you can a call flow, attach a call flow to a client, risk location or account. Refresh Call flow enables you to update the call flow with the latest information.

You use frames, attributes and rules to create a call flow. The Call Flow Editor is a visual editor you use to create the frames, attributes and rules that make a call flow.
