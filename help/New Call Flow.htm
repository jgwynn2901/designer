New Call Flow

1. From the Navigation bar, choose Hierarchy.

2. Click the "+" sign next to FNS in the Hierarchy tree.

3. Select the client in the Hierarchy Tree to go to the Business Entity Summary page in the right frame.

4. Click the "+" sign next to the Account Call Flows on the Business Entity page.

5. Click "New" on the Account Call Flows bar to go to the Call Flow page.

6. Enter or select Call Flow information below in the Call Flow and Account Call Flow areas.

  • Enter Name.

  • If applicable check Catastrophe.

  • Enter Description and select Line Of Business from drop down list.

  • If applicable check Call Start Flag.

7. Click the Save button.

8. In the call flow frames are right click on the call flow.

9. Select Visual Editor on the popup menu.

New Call Flow Frame appears.

See "New Frame" in the help for details on completing the New Call Flow Frame page. After completing the New Call Flow Frame page, see "Call Flow Editor" in the help for details on using the Call Flow Editor to create frames, attributes and rules to create a call flow.

Notes

Re-Assign Call Flow

Click the ReAssign Call Flow icon on the call flow page to re-assign the call flow to a client, risk location or account. Enter the call flow search information below.

  • Enter Name and select Line of Business from drop down list.

  • Enter CFID and Description.

  • Select Type of Search.

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click Search.

  • Select Result from results list.

  • Click the Select button.

 

Reference Count

The Reference Count is located in the Call Flow area on the Call Flow page. The reference count is the number of clients, accounts or risk locations that refer to or share the call flow. For example, a reference count of three means this call flow is shared with three other clients, accounts or risk locations. Click the Reference Count icon to display a Reference Count Report and see details.

 

Status Report

The Status report icon is located in the Call Flow area on the Call Flow page and displays messages for the call flow. For example "Warning! Shared Count is greater than 1." This message cautions you to be careful with this call flow because it is referred to or shared by other accounts, risk locations and clients.

 

Attach Rule

Click the Attach Rule icon in the Account Call Flow area on the Call Flow page to attach a rule to the call flow. Enter the Rule Search information below.

  • Select Type Of Rule from drop down list.

  • Enter Rule Text and Rule ID.

  • Select Type Of Search

The default is "Contains". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the Search button.

  • Select Result from results list.

  • Click the Select button.

 

De-Attach Rule

Click the De-Attach Rule icon in the Account Flow area on the Call Flow page to deattach the rule from the call flow.
