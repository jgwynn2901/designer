Escalation Search and New Escalation

If you need to find a particular Escalation Plan, you may want to use Escalation Search (Escalation Search examines the Escalation table for an Escalation Plan.). You do an Escalation Search to find the Escalation Plan and copy the plan to use in your call flow.

You can also use New Escalation (A new escalation is an original escalation added to the escalation table.) if you can not find an Escalation Plan to meet your needs.

Click Navigation Details for information on how to get to Escalation Search and New Escalation.

Navigation Details

1. From the navigation bar, choose "Maintenance."

2. Click the "+" sign next to the "Account Related" folder.

3. Select the "Escalation Plan" page on the Maintenance tree.

Common Tasks

These Common Tasks are used when using Escalation Search and New Escalation. Click the green text for detailed information on each task.

Escalation Search

Escalation Search

Enter the Escalation Search information.

  • Enter Escalation ID and A.H.S. ID.

  • Select Line of Business.

  • Select Type of Search.

The default is "Begins With." "Begins With" means the Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the "Search" button.

  • Select results from results list and click the "Details" tab.

  • Click the "Make Copy" button.

Note

You can use the "Back" button to return to the Escalation Search page.

New Escalation

New Escalation

  • Click the "New" button on the Escalation Search page.

  • Use AHS Search to attach Escalation Plan to account and find the AHS ID.

  • Select Line of Business and enter Description.

  • Click the Save button.

Note

You can use the "Back" button to return to Escalation page.
