Call Flow Search and New Call Flow

If you need to find a particular call flow, you may want to use Call Flow Search (Call Flow Search examines the call flow table for a call flow.). You use new call flow (A new call flow is an original call flow added to the call flow table.) to add a new call flow.

Click Navigation Details for information on how to get to Call Flow Search and New Call Flow.

Navigation Details

1. From the navigation bar, choose "Maintenance."

2. Click the "+" sign next to the "Call Flow Related" folder.

3. Select "Call Flow" page on the Maintenance tree.

Common Tasks

These Common Tasks are used when using Call Flow Search and New Call Flow. Click on the green text for detailed information on each task.

Call Flow Search

Call Flow Search

Enter the Call Flow Search information.

  • Enter Name and select Line of Business.

  • Enter CFID, Description and Frame Name.

  • Select Type of Search

The default is "Begins With." "Begins With" means the Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

Note

See "Building Call Flows" for details on the Call Flow page.
