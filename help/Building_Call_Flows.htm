Building Call Flows

You have three choices for building call flows: Standard Call Flow, Sharing Call Flows and New Call Flows.

Standard Call Flows

The best way to build a call flow is to use Standard Call Flows (A standard call flow is a pre-constructed call flow.). Standard Call Flows are similar to "templates" in Microsoft Word containing pre-made content that can be modified to meet your needs. For example, the "Standard Workers Compensation Call Start" call flow named "zSC WOR CALL START" requests the information required to correctly file a Workers Compensation Claim.

Standard Call Flows are time savers because a standard call flow has most of the information you'll need to build a call flow. You copy the standard call flow to the clipboard and paste the standard call flow from the clipboard to its new location. Because it is a copied call flow (A copied call flow is a duplicate of a call flow in the call flow table.) you can change the standard call flow by editing it.

Sharing Call Flows

If you plan to use the call flow as is, you may want to use a shared call flow (A shared call flow is an existing call flow in the call flow table.).

Sharing call flows has an important consequences. For example if you are sharing a call flow such GBS WOR you are connecting to the GBS WOR call flow in the call flow table. If you make changes to this shared call flow you are changing the call flow in the call flow table. Therefore you should use a shared call flow only if you can use the shared call flow as is. A shared call flow such as a GBS WOR should already have all the information required so you can use it without changing it.

You attach call flows (An attached call flow is a call flow joined to a call flow in the call flow table.) to share call flows. If you change your mind you can also detach call flows (A detached call flow is a call flow not joined to a call flow in the call flow table.).

New Call Flows

If your call flow requires a large amount of original information not existing on other call flows, you may want to use a new call flow (A new call flow is an original call flow added to the call flow table.).

New Call Flow enables you to create a new call flow from scratch. This allows you to create a call flow that exactly meets your needs.

Finding Standard Call Flows

You use "Call Flow Search" to find the standard call flow you need. Click "Navigation Details For Call Flow Search" for information on how to get to Call Flow Search.

Navigation Details For Call Flow Search

1. From the Navigation bar, choose "Maintenance" and click the "+" sign next to "Call Flow Related" on the Maintenance tree.

2. Click "Call Flow" to display the "Call Flow Search" page.

Enter or select the Call Flow Search information.

  • Enter Name and select Line of Business.

For example enter "zSC WOR CALL START" in Name and select "Workers Compensation" for LOB.

  • Enter CFID and Description.

For example enter "Standard Workers Compensation Call Start" in Description.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.Click "Search" and "Select" from results list.

  • Click Search.

  • Select Results from list.

Building Call Flows Using Shared Call Flows

Building Call Flows Using Shared Call Flows

You attach call flows (An attached call flow is a call flow joined to a call flow in the call flow table.) to share call flows. If you need to attach a call flow to a client, risk location or account you may want to attach call flow ( An attached call flow is a call flow joined to a client, risk location or account.). For example if you want to attach a call flow to Gallagher Bassett you would use Attaching Call Flows.

Click Navigation Details for information on how to get to Building Call Flows Using Shared Call Flows.

Navigation Details

1. From the Navigation bar, choose Hierarchy.

2. Click the "+" sign next to FNS in the Hierarchy tree.

3. Select the client in the Hierarchy Tree to go to the Business Entity Summary page in the right frame.

4. Click the "+" sign next to the Account Call Flows on the Business Entity page.

Attaching Call Flows

Click the "Attach" button on the Account Call Flows bar to go to Call Flow Search.

Enter the Call Flow Search information below.

  • Enter Name and select Line of Business from drop down list.

  • Enter CFID and Description.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click Search.

  • Select Results from list.

The Selected Result is attached to the client, risk location or account.

Building Call Flows Using New Call Flows

New Call Flow enables you to create a new call flow from scratch. This allows you to create a call flow that exactly meets your needs.

Click Navigation Details for information on how to get to Building Call Flows Using New Call Flows.

Navigation Details

1. From the Navigation bar, choose Hierarchy.

2. Click the "+" sign next to FNS in the Hierarchy tree.

3. Select the client in the Hierarchy Tree to go to the Business Entity Summary page in the right frame.

4. Click the "+" sign next to the Account Call Flows on the Business Entity page.

New Call Flows

Click "New" on the Account Call Flows bar to go to the New Call Flow page.

Enter or select Call Flow information in the Call Flow and Account Call Flow areas.

  • Enter Name.

  • If applicable check Catastrophe.

  • Enter Description and select Line Of Business from drop down list.

  • If applicable check Call Start Flag.

  • Click the Save button.

In the Call Flow Frames area the new call flow appears with a name and number.

Notes

Reference Count

The Reference Count icon tells you the reference count (The reference count is the number of clients, accounts or risk locations referencing the call flow.) for the call flow. The number to the right of the Reference Count icon is the reference count. For example, tells you nine other clients, accounts or risk locations are sharing this call flow.

You can also click the Reference Count icon to view a Reference Count Report and see detailed information on the references. The Reference Count icon appears at the top of the Call Flow section.

 

Status Report

The Status Report icon gives you a warning if the shared count is greater than one for the call flow. This messages cautions you to be careful with this call flow because it is shared by other accounts, risk locations and clients. For example, tells you more than one account, risk location or client is sharing this call flow.

You can also click the Status Report icon to view a Status Report and see detailed information on the warnings. The Status Report icon appears at the top of the call flow section.

 

Refresh

The Refresh icon updates the Designer with the latest information. It appears in the Call Flow Frames section.

 

Attach and Detach Rule to Call Flow

Attach Rule

Click the Attach Rule icon in the Account Call Flow section to attach a rule to the call flow. Enter the Rule Search information below.

  • Select Type Of Rule from drop down list.

  • Enter Rule Text and Rule ID.

  • Select Type Of Search

The default is "Contains". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the Search button.

  • Select Result from results list.

  • Click the Select button.

 

De-Attach Rule

Click the De-Attach Rule icon in the Account Flow section to detach the rule from the call flow.

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.
