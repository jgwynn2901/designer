Layout Bookmark Test

1. Select Frame from the Frame drop down list on the Call Flow Editor.

2. Click the "Layout" tab on the Call Flow Editor.

3. Click the Attribute Search icon on the Layout tab bar.

4. Either find an attribute to attach to the frame or create a new attribute to attach to the frame.

Click Find Attribute to see Find Attribute Information.

To Find Attribute

  • Enter Name and Caption.

  • Select Input Type from drop down list.

  • Enter Description, Help String and Attribute ID.

The Help String is information in the Tips Frame in the call flow screen. The Tips Frame contains suggestions for taking the information from the caller when the call representative uses the attribute.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the Search button.

  • Select appropriate result from results list.

  • Click the Select button.

The attribute appears in the Attribute Name List drop down box.

 

To Create New Attribute

  • Click the New button on the Attribute Search dialog box.

  • Enter Name and Caption.

  • Enter Unknown Value and select Input Type from drop down list.

  • Enter Text Length, Entry Mask and Default Value.

  • Enter Description and Help String.

The Help String is information in the Tips Frame in the call flow screen. The Tips Frame contains suggestions for taking the information from the caller when the call representative uses the attribute.

You can attach or detach rules to the attribute. See "Attribute Rules" in the help for details. You can also Attach Lookup Type or Detach Lookup Type. See "Lookup Types" in the help for details.

5. Click the Save button.

Notes

The Call Flow Editor has several Layout Tab Options and Frame Popup Menu Options you can use after you added the attribute to the frame. There are some many options you can use I will not list them all.

 

Layout Tab Options

Instance Properties

Changes the properties of an instance. See "Instance Properties" in the help for more information.Notes Overview

Changes the properties of an attribute. See "Attribute Properties" in the help for more information.

Over Ride

Overrules an attribute property. See "Over Ride" in the help for more information.

Remove

Removes the attribute from the frame. Click the Remove icon on the Layout tab or right click the mouse inside the attribute and select Remove from popup menu. Notes Overview

 

 

Undo

Undoes the last action. Click the Undo icon on the Layout tab or right click the mouse inside the frame and select Undo from the popup menu. Notes Overview

Save

Saves any changes in the Designer. Click the Save icon on the Layout tab or right click the mouse inside the attribute and select Save from the popup menu. Notes Overview

Zoom

Zooms in for close or enlarged view of the the frame. Click the Zoom icon on the Layout tab or right click the mouse inside the frame and select "Zoom" from the popup menu.

 

Frame Popup Menu Options

Show Select

Displays the attribute selected. Right click the mouse inside the frame and select "Show Select" from the popup menu.

Show All

Displays all the attributes on the frame. Right click the mouse inside the frame and select "Show All" from the popup menu.

Show Attributes

Displays attributes such as "CLAIM:CLAIM_NUMBER". Right click the mouse inside the frame and select "Show Attributes" from the popup menu.

Show Samples

Displays the attributes as they would appear to the user. Right click the mouse inside the frame and select "Show Samples" from the popup menu.
