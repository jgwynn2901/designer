Instance Properties Drop Down Hotspot

An Instance is an occurrence of something. Instance Properties contains the characteristics of the Instance. When you use an Instance, you are essentially using a copy of the attribute from the attributes database. If you use Instance Properties and make changes the changes effect only your instance of the attribute not the original attribute in the attributes database.

Click Navigation Details for directions to locate Instance Properties on the Designer.

Navigation Details

1. Click the Properties icon on the Layout tab or right click the mouse inside the attribute.

2. Select Instance Properties from the popup menu.

3. Enter or select Instance Properties information below.

  • Enter Caption and Sample Value.

  • Enter Xpos, Y Pos, Width, Height, Sequence and select Auto Type.

  • Enter LU Column Name and LU Storage Name.

  • If applicable, check LU Display flag, LU Storage Flag, Mandatory, Valid Value and Spell Check.

  • Enter Entry Mask and Default Value.

  • Enter Unknown Value and Text Length.

  • Enter Help String and Description.

The Help String is information in the Tips Frame in the call flow screen. The Tips Frame contains suggestions for taking the information from the caller when the call representative uses the attribute.

  • Click the OK button.

Notes

You can attach or detach rules to the attribute. See "Attribute Rules" in the help for details. You can also attach lookup types or detach lookup types. See "Lookup Types" in the help for details. In addition you can Attach Existing Frame To This Call Flow or Detach this Frame From Current Call Flow. See "Attach Existing Frame To This Call Flow" or "Detach This Frame From Current Call Flow" in the help for more information.
