Adding Output Items

Output Items are composed of output definitions and output pages.

You have four choices in Adding Output Items. If your Output Item requires a great deal of original information found on other output definitions, you may want to use New Output Item Using Attached Output Definition (New output item using attached output definition references an output definition in the output definitions database.). If your output item requires a great deal of original information not found on other output definitions, you may want to use New Output Item Using New Output Definition (New output item using new output definition creates an original output definition added to the output definitions database.).

If you want to update output item properties information, you may want to use Output Item Properties (Output Item Properties enables you to edit output item properties.). If you want to erase the output item use Delete Output Item (Delete output item deletes the output item.).

Click Navigation Details for information on how to get to Adding Output Items.

Navigation Details

1. From the Navigation bar, choose Hierarchy.

2. Click the "+" sign next to FNS in the Hierarchy tree.

3. Select the client in the Hierarchy Tree to go to the Business Entity Summary page in the right frame.

4. Click the "+" sign next to the Routing Plans on the Business Entity page.

5. Select routing plan from list of routing plans and click "Edit" on the Routing Plans bar.

Common Tasks

These Common Tasks are used when Adding Output Items. Click the green text for detailed information on each task.

New Output Item Using Attached Output Item

New Output Item Using Attached Output Item

Right click the mouse on the transmission sequence step on the routing plan in the Routing Plan details section and select "Create Output Item" from popup menu.

  • Click the "Attach Output Definition" icon to display the Output Definition Search dialog box.

  • Enter Name, OD ID and Description.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the Search button.

  • Select the appropriate result from results list.

  • Click the "Select" button.

The Output Definition is attached to the output item and the output definition ID appears in the Output Definition box.

  • Enter Sequence.

  • Click the "Save" and "Close" buttons.

The Output Item appears in the Routing Plan Details area under the transmission sequence step.

New OutPut Item Using New Output Definition

New Output Item Using New Output Definition

Right click the mouse on the transmission sequence step on the routing plan in the Routing Plan details section and select "Create Output Item" from popup menu.

Enter or select New Output item information.

  • Click the "Attach Output Definition" icon to display the Output Definition Search dialog box.

  • Click "New" on the Output Definition Search dialog box.

  • Enter Name and Description.

  • Click "Save" and "Close."

The Output Item appears in the Routing Plan Details area under the transmission sequence step.

Output Item Properties

Output Item Properties

Right click the mouse on the Output Item on routing plan in the Routing Plan details section and select "Output Item Properties" from the popup menu.

Update or change Output Item Properties Properties. Click "Save" and "Close."

Delete Output Item

Delete Output Item

Right click the mouse on the Output Item on routing plan in the Routing Plan details section and select "Delete Output Item" from the popup menu.
