Routing Plan Components

Routing Plan components are Transmission Sequence Steps, Output Items, Output Definitions and Output Pages.

Transmission Sequence Steps are used to direct the transmission. It tells the how and where information. In the example below the "how" is "Print" and "where" is "OPER_HPSI."

Example

Output Items are what is sent. Output Items are output pages such as a cover letter, first report of injury and overflow page.

Example

Output Definitions are used to create output items such as output pages and overflow pages.

Example

One type of output page is the First Report Of Injury form. These forms consist of a background image and a series of fields mapped on to that image.

Example

Another type of output page is the overflow page. The overflow page is used to capture information taken during the call that doesn't fit on the First Report of Injury form.
