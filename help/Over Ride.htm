Over Ride

Over Ride enables you to overrule or change an attribute property. You can create a new attribute over ride, edit attribute over ride or delete attribute over ride.

New Attribute Over Ride

1. Click the Over Ride icon on the Layout tab or right click inside the attribute and select Over Ride from the popup menu.

2. Select New on the Attribute Over Ride Listing bar.

3. Enter or select New Attribute Over Ride information below.

  • Select Property Name from drop down list.

  • Enter Sequence.

  • Click the Attach Rule icon to find a rule to attach to the override or create a new rule to attach to the over ride.

To Find Rule

  • Select Type.

  • Enter Rule Text and Rule ID.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the Search button.

  • Select result from results list.

  • Click the Select button.

To Create New Rule

  • Click the New button on the Rule Search box.

  • Select Rule Type from drop down list.

  • Enter Rule Text.

  • Click the Save button.

  • Enter Attribute Over Ride data.

    • 4. Click Save button on the Attribute Over Ride Details dialog box.

    Edit Attribute Over Ride

    1. Click the Over Ride icon on the Layout tab or right click inside the attribute and select Over Ride from the popup menu.

    2. Select the appropriate attribute over ride.

    3. Click the Edit button on the Attribute Over Ride Listing box.

    4. Edit or change the Attribute Over Ride Information below.

    • Property Name

    • Sequence

    • Over Ride Rule

    • Attribute Over Ride Data

    5. Click the Save button.

    Delete Attribute Override

    1. Click the Over Ride icon on the Layout tab or right click inside the attribute and select Over Ride from the popup menu.

    2. Select the appropriate attribute over ride.

    3. Click the Delete button on the Attribute Over Ride Listing box.

    4. Click OK on the "Are you sure you want to delete this over ride?" dialog box.
