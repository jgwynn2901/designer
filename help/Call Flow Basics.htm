Call Flow Basics

Call Flow Basics contains the fundamentals of call flows. These topics provide the essentials of building call flows as well as call flow components in step by step procedures.

If you need to know where to begin to build call flows read "Building Call Flows." You may want to add call flow components such as frames, attributes and rules. "Adding Frames to Call Flows" describes the options you have for adding frames to call flows. "Adding Attributes to Frames" describes the alternatives for adding attributes to frames. You have several options for adding rules to call flows, frames and attributes as described in "Adding Rules to Call Flows, Frames and Attributes."

"Adding Frame Orders" details how to order the frames in a call flow. "Reviewing Attribute Instance Listing and General Information" provides attribute instance and general information.

You have several choices at the call flow level. "Editing and Refreshing Call Flows" explains how to edit and refresh call flows. "Copying and Pasting Call Flows" details how to copy and paste call flows. "Filtering and Deleting Call Flows" details how to filter and delete call flows. "Attaching Call Flows" tells about attaching call flows.
