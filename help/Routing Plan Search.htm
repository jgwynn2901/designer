Routing Plan Search

1. From the Designer Navigation bar, choose Hierarchy.

2. Click the "+" sign next to FNS in the Hierarchy Tree.

3. Select a client in the Hierarchy Tree.

4. Right click the mouse on the client.

5. Click "Node Search" on the popup menu.

6. Select Routing Plan tab.

7. Enter or select Routing Plan Search information below.

  • Enter Destination Type, select Input System Name and Line Of Business, enter Routing Plan ID, Description, select state or any combination.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Select Direction

The default is "Down". "Down" means Designer searches down the path to the bottom of the Hierarchy Tree. "Up" means Designer searches up the parent path to the top of the hierarchy tree.

8. Click the Search button.

Notes

Go To Node

The Go To Node button goes to the Client Node on the Hierarchy Tree and displays the Business Entity Summary page for the selected result. To use the Go To Node button:

  • Select appropriate result from results list.

  • Click the "Go To Node" button.

 

Copy

The Copy button copies the result to the clipboard so you can paste it later. To use the Copy button:

  • Select appropriate result from results list.

  • Click the "Copy" button.

 

Close

The Close button closes the Search dialog box.

 

Clear

The Clear button clears the search criteria.

 

Partial Searches

You can do partial searches. For example, entering the letters "Smit" in Name and clicking Search returns a list of names that begin with the letters "Smit" such as "Smith" and "Smits".

It is suggested you enter as much information as possible when doing a partial search. This refines the search and limits the number of search results. In addition, a refined search produces faster search results.
