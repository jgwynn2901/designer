Attribute Properties

An attribute is a field such as "Agent Zip Code" or "Previous Caller Lookup." Attribute Properties contains characteristics of the attribute. When you use an attribute, you are using an existing attribute from the attributes database. If you use Attribute Properties and make changes you are changing the attribute in the attributes database. The attribute in the attributes database is most likely used in other call flows. Therefore use this option only if you want to use the attribute as is.

To use Attribute Properties

1. Right click the mouse inside the attribute.

2. Select Attribute Properties from the popup menu.

3. Enter or select Attributes Properties information below.

  • Check the Edit box.

  • If applicable check Valid Value? and Spellcheck?

  • Enter Unknown Value and select Input Type from drop down list.

  • Enter Text Length, Entry Mask and Default Value.

  • Enter Description and Help String.

The Help String is information in the Tips Frame in the call flow screen. The Tips Frame contains suggestions for taking the information from the caller when the call representative uses the attribute.

  • Click the Save button.

Notes

You can attach or detach rules to the attribute. See "Attribute Rules" in the help for details. You can also Attach Lookup Type or Detach Lookup Type. See "Lookup Types" in the help for details.
