Output Page Search and New Output Page

If you need a particular output page, you may want to use Output Page Search (Output Page Search examines the output page table for a output page.). You use New Output Page (A new output page is an original output page added to the output page table.) to add a new output page.

Click Navigation Details for information on how to get to Output Page Search and New Output Page.

Navigation Details

1. From the navigation bar, choose "Maintenance."

2. Click the "+" sign next to the "Routing Plan Related" folder.

3. Select "Output Page" page on the Maintenance Tree.

Common Tasks

These Common Tasks are used when using Output Page Search and New Output Page. Click the green text for detailed information on each task.

Output Page Search

Output Page Search

Enter Output Page Search information.

  • Enter Name and Output Definition ID.

  • Enter BMP and Output Page ID.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the "Search" button.

  • Select the result from the results list and click the "Details" tab.

  • Click the "Make Copy" button.

Note

You can use the "Back" button to return to the Output Page Search page. You can also use the "Delete" button to delete the output page. In addition you can use the "Visual Editor" button to see the output page.

New Output Page

New Output Page

Click the "New" button on the Output Page page.
