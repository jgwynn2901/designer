Attach Existing Frame To This Call Flow

1. Click the Attach Existing Frame To This Call Flow icon in the upper left corner of the Call Flow Editor.

2. Enter or select Frame Search information below.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click the Search button.

  • Select the appropriate result from the results list.

  • Click the Select button.

3. Click the Save button.

Note

The attached frame appears in the Call Flow Frames drop down list located at the top of the Call Flow Editor. You can also minimize the Call Flow Editor to see the attached frame in the Call Flow Frames area.
