Context Sensitive Help

Context Sensitive Help refers to help you access from within an application. It provides a specific help topic where you want to find it. Context Sensitive Help displays a help topic that is directly relevant to the task you are trying to accomplish.

This automatic response eliminates the need for you to wade through other methods of topic access (Table Of Contents, Index) to find the appropriate help topic. You get the appropriate help topic quickly and without having to search for it.

Designer Help features window level context sensitive help. Window level context sensitive help provides an explanation of all items in a window or screen within one help topic.

How To Use Context Sensitive Help

The Designer provides context sensitive help using a help icon. Click the help icon on the page to display the appropriate help topic.

Let's say, for example, you are using the Business Entity Summary page and have a question about it. Click the help icon on the "Business Entity Summary" page.

 A help window appears in the upper right corner containing the "Business Summary" help topic.

Or perhaps you have a question on the Account Call Flow Section of the Business Entity Summary page. Click the help icon next to "Account Call Flow".

A help window appears in the upper right corner containing the "Account Call Flows" topic.
