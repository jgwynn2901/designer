Filtering and Deleting Call Flows

You have several choices at the call flow level on the Business Entity Summary page. You can filter call flows or delete call flows.

If you need a customized list of call flows, you may want to filter call flows (A filter is a device that screens information passing certain information and blocking certain information. Filter call flows creates a refined list of call flows based on the criteria you entered.). For example, if you wanted a list of Worker Compensation call flows you would use filter call flows.

If you need to remove a call flow, you may want to delete call flows (A deleted call flow is removed from the call flow database.).

Click Navigation Details for information on how to get to Filtering and Deleting Call Flows.

Navigation Details

1. From the Navigation bar, choose Hierarchy.

2. Click the "+" sign next to FNS in the Hierarchy tree.

3. Select the client in the Hierarchy Tree to go to the Business Entity Summary page in the right frame.

4. Click the "+" sign next to the Account Call Flows on the Business Entity page.

Common Tasks

These Common Tasks are used when Filtering and Deleting Call Flows. Click the green text for detailed information on each task.

Filtering Call Flows

Filtering Call Flows

Click the "Filter" button on the Account Call Flows bar.

Enter the Call Flow Search information below.

  • Enter Name and select Line of Business from drop down list.

  • Enter CFID and Description.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. "Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

  • Click Search.

Deleting Call Flows

Deleting Call Flows

Select call flow from call flows list.

Click the "Delete" button on the Account Call Flows bar.

Click "OK" on the "Are you sure you want to remove Account Call Flow ID?" dialog box.
