Add Child Node

Add Child Node adds a node to a parent node. This enables you to add clients, risk locations and accounts.

Click on Navigation Details for information on how to get to Add Child Node.

Navigation Details

1. On the Designer navigation bar, click Hierarchy.

2. Click the "+" sign next to FNS on the Hierarchy Tree.

3. Select a client, risk location or account on the Hierarchy Tree.

  • Right click the mouse on the client, risk location or account.

  • Click "Add Child Node" on the popup menu.

  • Enter or select Add Child Node information.

  • Click the Save button.

Notes

Status Report

The Status report icon is located in the Call Flow area on the Call Flow page and displays messages for the call flow. For example "Warning! Shared Count is greater than 1." This message cautions you to be careful with this call flow because it is referred to or shared by other accounts, risk locations and clients.
