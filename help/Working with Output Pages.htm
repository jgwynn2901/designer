Working with Output Pages

Output Pages are cover letters, First Report of Injury and Overflow. Output pages have two parts. The underlying bitmap image is set on the "General" tab in the Output Definition Editor. The attributes mapped to that image are set in the "Layout" tab in the Output Definition Editor. You can view the attributes using the "layout" tab or see a list of attributes using the "Listing" tab.

You have two options concerning attributes. You can find attributes and map them to forms or create new fields from existing attributes (for example, an address field made up of street, city, state and zip code attributes).

Find Attributes & New Output Fields

If your attribute requires a great deal of the same information found on other attributes, you should use Find Attributes (Find Attribute is an attribute search based on criteria you enter.) to use an existing attribute.

New Output Fields adds output fields to the Output page. You can also add fields that are composites of several attributes, or that execute conditional logic related to their appearance on the form.

Output Field Properties

Output Field Properties enables you to update or change Output Field properties. Output Field properties are used to combine and format different attributes. It is also used to set rules for when to show fields. For example to show a field if a certain attribute equals a certain value. Otherwise show something else or show nothing.

Layout Tab Icons and Popup Menu

The Layout Tab contains icons you use when adding attributes. These icons are Attribute Search, Output Field Properties, Save, Undo and Zoom. The Popup Menu contains choices that effect the way you view information on the Layout Tab. These choices are Show Select, Show All, Show Attributes and Show Samples.

Find Attributes

Find Attributes

Click the Attribute Search icon on the Layout tab bar.

  • Enter Name and Caption.

  • Select Input Type from drop down list.

  • Enter Description, Help String and Attribute ID.

The Help String is information in the Tips Frame in the call flow screen. The Tips Frame contains suggestions for taking the information from the caller when the call representative uses the attribute.

  • Select Type of Search

The default is "Begins With". "Begins With" means Designer returns results beginning with the criteria you entered. ""Contains" means Designer returns results containing the criteria you entered. "Exact" means Designer returns results with the exact criteria you entered.

Layout Tab Icons and Popup Menu

Layout Tab Icons and Popup Menu

Layout Tab Icons

  • Attribute Search icon accesses attribute search.

  • Properties icon accesses Instance Properties or Attribute Properties.

  • Save icon saves any changes to the Designer.

  • Undo icon undoes the last action.

  • Zoom icon zooms in for a close or enlarged view of the frame.

  • Override icon overrides an attribute property.

  • Remove icon removes the attribute from the frame.

Popup Menu

Show Select

Displays the attribute selected. Right click the mouse inside the frame and select "Show Select" from the popup menu.

Show All

Displays all the attributes on the frame. Right click the mouse inside the frame and select "Show All" from the popup menu.

Show Attributes

Displays attributes such as "CLAIM:CLAIM_NUMBER". Right click the mouse inside the frame and select "Show Attributes" from the popup menu.

Show Samples

Displays the attributes as they would appear to the user. Right click the mouse inside the frame and select "Show Samples" from the popup menu.
