Installation troubleshoot
The Designer Login page failed to installed the latest controls necessary to run the Designer application. Please print this page and close all instances of Internet Explorer before applying the following steps.

To correct this problem, do the following:

  • Close this Internet Explorer browser window.
  • From the Start menu, select Settings | Control Panel.
  • Double-click the Internet icon to launch the Internet Properties dialog.
  • Under Temporary Internet Files, click on the Settings button.
  • In the Settings dialog, click on View Objects.
  • Right-click on each of the following items and select the Remove menu command: FNSTree.ctlDynTree, ClipboardAgent.Class, LayoutCtl.Class
  • If it prompts you to remove other files, click 'yes' to all of them.
  • When all the above items are removed, close out all Settings dialogs by clicking OK.
  • Open up the Internet Explorer browser and run the Designer Login page again. It will prompt you to install the newer components. Click 'yes' to continue. Reboot your PC if requested.
  • If you still have problems, repeat the above steps and then:
  • Locate these DLLs files in your windows/system directory: Dyntree.ocx, FNSTree.ocx, FNSDClipboard.dll and FNSPageEditor.dll
  • Manually unregister these DLLs using the regsvr32.exe utility:
  • Example: regsvr32.exe /u FNSDClipboard.dll
  • Open up the Internet Explorer browser and run the Designer Login page again. It will prompt you to install the newer components. Click 'yes' to continue.
